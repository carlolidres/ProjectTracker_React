## Summary

Creatable searchable autocomplete for key project/support/CNF fields, new RnD user role, compact document status icons in CNF detail tables, and Dashboard My Worklist as a wide searchable modal with Process/Support tabs and role-scoped defaults.

## Features added

- Creatable registry selects: `project_owner`, `client_name`, `activity_type`, `uom`, `business_unit` (Project Entry + Projects Database)
- Support creatable options: department (registry), line/material/principal/product (`reusable_options`)
- CNF Initiator creatable via `reusable_options` category `cnf_initiator`
- User role `rnd` (RnD) — enum migrations + menu defaults (Support-focused)
- CNF detail PO/support document numbers show status icons with accessible tooltips
- Dashboard My Worklist modal: Process/Support tabs, All Worklist toggle (default Off), global search, no pagination

## Security / configuration changes

- Supabase: `20260719120000_rnd_role_enum`, `20260719120100_rnd_role_helpers` (additive enum value + role helpers)
- Registry/reusable option creates remain subject to RLS; option delete soft-removes suggestions only

**Risk:** Low–Medium. Rollback frontend: redeploy `0.91.0` @ `c264715`. Do not drop enum value `rnd` without a planned reverse.
